SpEL Validator

基于 SpEL 的 Java 参数校验器

快速开始项目介绍

易上手

设计直观，简单易用，与 javax.validation 使用方式高度相似，学习成本低，上手快。

极致灵活

利用 SpEL（Spring Expression Language）的强大功能，轻松实现复杂验证逻辑，且能够直接调用已注入的 Spring Beans 进行验证。

无缝集成

扩展自 javax.validation 包，只新增不修改，无缝集成到现有项目中。

上下文感知

能够在整个对象的上下文中进行验证，支持跨字段的逻辑验证，提高数据完整性检查的效率和准确性。

异常友好

验证失败时自动整合进 javax.validation 的异常管理体系，简化错误处理流程。

完全可定制

支持创建自定义约束注解，允许根据具体业务需求灵活定义验证规则。
